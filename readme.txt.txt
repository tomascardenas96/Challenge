Pagina de registro e inicio de sesion con autenticacion:

INSTALACION:
-Clonar el repositorio: $ git clone https://github.com/tomascardenas96/Challenge.git
-Instalar Nest.js (En caso de no tenerlo): $ npm install -g @nestjs/cli
-Desde la terminal, instalar las dependecias de las carpetas Backend y Frontend posicionandote
sobre sus respectivos directorios y posteriormente ejectutar el siguiente comando para ambos casos: $ npm istall

PREPARACION:
-En la terminal, posicionarse sobre la carpeta "Backend" y enviar el comando $ npm run db.
-En una nueva terminal, tambien en la carpeta "Backend" enviar el comando $ npm run start:dev.
-En total necesitas tener 2 terminales abiertas, una para el json-server(Que simulara la base de datos) y otra para
levantar el servidor

-Para ejecutar el codigo, instalar la extension "Live server" en Visual studio code y
desde la carpeta raiz (Donde esta este mismo archivo de texto), presionar el boton "Go live" ubicado en
la esquina inferior derecha de VSC.
Al hacer click en dicho boton, se abrira el navegador web dandote la eleccion de seleccionar una carpeta,
seleccionar la carpeta Frontend para poder ver la aplicacion.
